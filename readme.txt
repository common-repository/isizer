Simple change, create and remove registered WordPress image sizes

== Description ==

Simple change, create and remove registered WordPress image sizes in one (maybe two) clicks.



== Installation ==

1. Upload `isizer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
